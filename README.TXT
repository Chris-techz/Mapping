								***** Robot Mapper V1 *****

Le logiciel permet a travers une IHM, de remplir un fichier texte de coordonnées que l'on va envoyer au robot.

Il est possible de rentrer des zones "MORTE" (noir) et des zones "ITINERANTE" (rouge). Les cases blanches sont "LIBRE".

Il faut avant tout rentrer les dimensions du lieu ou évolue le(s) robot(s) (Longueur / Largeur), le logiciel vous indique ensuite la donnée métrique par case.

Le boutton "Effacer Tableau" va effacer le contenu du tableau, mais également le fichier de coordonnées (Attention !).
